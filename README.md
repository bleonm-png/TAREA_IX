# README - Sistema de Registro de Usuarios en C++

Este programa permite **guardar y mostrar usuarios** utilizando **archivos de texto** en C++.

## Funciones del programa

El sistema muestra un menú con tres opciones:

1. **Guardar usuario:** solicita el nombre completo, la posición en la empresa y la edad del usuario, luego guarda los datos en un archivo llamado `usuarios.txt`.
2. **Mostrar usuarios:** lee y muestra en pantalla todos los usuarios almacenados en el archivo.
3. **Salir:** finaliza la ejecución del programa.
